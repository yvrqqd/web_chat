# web_chat
Клиент/серверное приложения для обмена сообщениями между клиентами. 
- ЯП Python >= 3.8, asyncio для сетевого программирования 
- Протокол TCP
- Формат сообщений JSON

Проверено на Win11 Python3.12

## Запуск
cd web_chat  
python client.py  
python server.py
